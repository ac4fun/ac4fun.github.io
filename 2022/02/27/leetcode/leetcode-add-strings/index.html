大数加法

发表于 2022-02-27 更新于 2022-02-28 分类于 leetcode 阅读次数： Valine：
本文字数： 683 阅读时长 ≈ 1 分钟

给定两个字符串形式的非负整数 num1 和num2 ，计算它们的和并同样以字符串形式返回。

你不能使用任何內建的用于处理大整数的库（比如 BigInteger）， 也不能直接将输入的字符串转换为整数形式。

 

示例 1：

输入：num1 = “11”, num2 = “123”
输出：”134”
示例 2：

输入：num1 = “456”, num2 = “77”
输出：”533”
示例 3：

输入：num1 = “0”, num2 = “0”
输出：”0”
