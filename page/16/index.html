编辑距离

发表于 2022-04-04 分类于 leetcode
本文字数： 1.3k 阅读时长 ≈ 1 分钟

给你两个单词 word1 和 word2， 请返回将 word1 转换成 word2 所使用的最少操作数  。

你可以对一个单词进行如下三种操作：

插入一个字符
删除一个字符
替换一个字符  

示例 1：
输入：word1 = “horse”, word2 = “ros”
输出：3
解释：
horse -> rorse (将 ‘h’ 替换为 ‘r’)
rorse -> rose (删除 ‘r’)
rose -> ros (删除 ‘e’)

示例 2：

输入：word1 = “intention”, word2 = “execution”
输出：5
解释：
intention -> inention (删除 ‘t’)
inention -> enention (将 ‘i’ 替换为 ‘e’)
enention -> exention (将 ‘n’ 替换为 ‘x’)
exention -> exection (将 ‘n’ 替换为 ‘c’)
exection -> execution (插入 ‘u’)

动态规划

对于两个串A和B，编辑动作可以归纳为三类：

  1. A后添加一个数字
  2. B后添加一个数字
  3. A修改一个数字

dp[i][j]表示A的前i个字符和B的前j个字符的编辑距离。
可以拆解为三个子问题：

  1. dp[i-1][j], A的前i-1个字符和B的前j个字符, 可以通过A后添加一个数字得到目标问题
  2. dp[i][i-1], 同理，可以在B后添加一个数字得到目标问题
  3. dp[i-1][j-1], A的前i-1个字符和B的前j-1个字符, 我们可以修改A的第i个字符使之和B的第j个字符一致得到目标问题。
    如果二者原本相同，则不需要修改，否则编辑距离+1。
